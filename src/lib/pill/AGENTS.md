# `@dvcol/neo-svelte/pill`

```ts
import { NeoPill } from '@dvcol/neo-svelte/pill';
```

## Components

- `NeoPill` — chip / tag with optional removable affordance and color tokens.

## Concepts

- `closable` adds a close button; bind `onclose` to remove the pill from the parent collection.
- Color follows the active palette (`primary`/`secondary`/`error`/...) or a custom token override.

See also: [badge](../badge/AGENTS.md), [buttons](../buttons/AGENTS.md).
