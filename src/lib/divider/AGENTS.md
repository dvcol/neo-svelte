# `@dvcol/neo-svelte/divider`

```ts
import { NeoDivider } from '@dvcol/neo-svelte/divider';
```

## Components

- `NeoDivider` — horizontal/vertical separator. Supports a `label` snippet for inline section titles.

## Concepts

- `orientation` switches axis; in flex contexts the divider stretches to the cross-axis automatically.
- Color/inset are token-driven (`--neo-divider-color`, `--neo-divider-inset`).
